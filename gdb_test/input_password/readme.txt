Компиляция:
gcc input_password.c -o input_password.out -g -fno-stack-protector -no-pie

Запуск: ./input_password.out < input_123.txt

Адрес возврата в main при корректной работе:
 0x4011b3 (8 байт).
Адрес команды, которая должна выполниться при корректном пароле (после else):
 0x4011d2.

Файлы input_123.txt и input_test.txt содержат пример некорректного
 и корректного паролей.
Файл input_hack.txt содержит "пароль" hackpassword,
 после которого перезаписывается rbp
  (используется адрес 0x00007fffffffe300, но можно и любой другой),
 и новый адрес возврата (0x00000000004011d2).
Адреса записываются в файл в обратном порядке.
